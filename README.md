# Draft Lottery

## Description

A simple vanilla JS app to run a draft lottery in the context for fantasy sports drafts.

## Tech Stack

Vanilla JS/HTML/CSS

## Usage Instructions

- Set the number of teams that are in the lottery using the input field
- (optional) Rename the teams and adjust the number of lottery balls each team has
- Click "Start Lottery" and the percentages of the remaining teams will appear
- Click "Draw Next Pick" until the number of remaining teams hits zero
